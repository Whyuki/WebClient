Attention : toute variable déclarée dans une fonction n'est utilisable que dans cettemême fonction.
Ces variables spécifiques à une seule fonction ont un nom : les variableslocales.
Déclarées en dehors des fonction, on parle de variables globales.

Mais on évite de créer des variables locales et globales qui portent le même nom.
Enrègle générale, on préfère utiliser des variables locales (pour éviter les confusions)
